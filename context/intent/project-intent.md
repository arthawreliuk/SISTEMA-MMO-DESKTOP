# Project Intent — SISTEMA-MMO-DESKTOP

## Identidade

- **Nome**: SISTEMA-MMO-DESKTOP
- **Produto**: Mestre da Mão de Obra
- **Tipo**: Marketplace de serviços profissionais (estilo GetNinjas)
- **Objetivo**: Protótipo interativo para demonstração ao cliente em reunião

## Tech Stack

- HTML5 (estático, sem servidor)
- Tailwind CSS (via CDN)
- Font Awesome 6
- Google Fonts (Plus Jakarta Sans)
- Vanilla JS (mínimo — apenas navegação entre telas)

## Público-alvo

| Perfil | Objetivo no sistema |
|---|---|
| Cliente | Contratar serviços profissionais |
| Profissional | Se vender / oferecer serviços |
| Empresa | Contratar profissionais para projetos |

## Escopo do Protótipo

**Foco**: interatividade visual entre telas — zero backend, zero banco de dados.

- Desktop: escopo atual (26 módulos)
- Tablet/Mobile: fase futura (mesma lógica de fluxo, arquivos separados)

## Deliverables Contratados

1. Protótipo HTML interativo (fluxo 100% linkado, todos os módulos)
2. Telas exportadas em PNG

## Definition of Done

- Fluxo completo entre todas as telas sem nenhuma tela intermediária faltando
- Qualquer elemento clicável leva para a tela correta
- Protótipo abre diretamente no browser (sem servidor local)
