# Intent: Feature — Linkagem Interativa das Telas (26 Módulos Desktop)

## What

Adicionar atributos de navegação (`href`, `onclick`) nos elementos clicáveis de todos os arquivos HTML dos 26 módulos desktop, conectando o fluxo completo entre as telas para que o protótipo funcione no browser sem backend.

## Why

O cliente Mestre da Mão de Obra precisa visualizar o produto em reunião com compartilhamento de tela. Sem linkagem, as telas são estáticas e não demonstram a experiência real de uso da plataforma.

## Constraints

- NUNCA alterar CSS/layout — apenas `href`, `onclick`, `id`
- Paths relativos obrigatórios (sem `http://` interno)
- Botões Google/Apple removidos das telas de auth (instrução explícita do cliente)
